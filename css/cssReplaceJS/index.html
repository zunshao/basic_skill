css 单位 rem,em,vh,vw,vmin,vmax
  • em:相对于父元素font-size
  • rem:r为root的意思，body元素font-size。@html元素上设置亦可，注意css样式查找原理
  • vh:当前viewport，高度height 1/100
  • vw:当前viewport，宽度width 1/100
  • vmin&vmax当前viewport，两都是取width/height值1/100。vmin取宽或高的最小值，vmax表示取宽或度的最大值
