### 域名解析 ###
+ 域名网站 解析至服务器IP即可
+ 域名解析时 前缀场景 www @ *
    + www 正常解析
    + @ 省略 www. 前缀解析
    + * 泛解析，即任意前缀即可 *.baidu.com
